# Buck converter of 24Vin DC votlage to 3.3Vout DC  voltage
This repository contain the circuit diagram of buck converter which convert the 24v input dc voltage to 3.3v dc output voltage by using AP64501SP ic.

## Hardware specification
- AP64501SP ic
- input decoupling capacitors for stable dc input voltage
- SMBJ36A diode
- onboard 2 pin header connector for input  voltage and output voltage connection
- Feedback resistors
- Output capacitors

## software 
- kicad 9.0
